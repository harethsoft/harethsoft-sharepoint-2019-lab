
## 5. Modern Team Sites Template – SharePoint 2019 

### ?? Lesson Objective

You want to know **how to create and use a Modern Team Site in SharePoint 2019** — not just theory, but **how to do it step-by-step**

---

###  What is a Modern Team Site?

A Modern Team Site is a ready-to-use site template in SharePoint 2019 that allows a team to collaborate, share documents, 
create lists, and work together with a **modern interface**.
 It is faster, cleaner, and mobile-friendly compared to classic sites.

---

### ?? When to use Modern Team Site

Use it when:

* You want a **team collaboration workspace**
* Multiple users will **share files and work together**
* You want a **modern experience (UI)**
* Classic sites are not needed anymore 

---

## ?? **How to Create a Modern Team Site (Step-by-Step)**

### 1. Go to SharePoint Central Administration

Open the SharePoint server > **Central Administration**.

---

### 2. Open Site Collections

Click on **Application Management** ? **Create site collections**.

---

### 3. Select Web Application

Choose the Web Application where the site will be created.

---

### 4. Choose the Template

Under **Template Selection**, scroll to find:
**Team Site (Modern Experience)** and select it.



---

### 5. Enter Details

Fill in:

* **Title** (Name your site)
* **URL** (the link users will open)
* **Primary Site Collection Administrator** (the main admin)

---

### 6. Save

Click **OK** to create the site.

---

## ?? What Happens After Creation

* A **Modern Team Site** is created.
* Default groups are provisioned:

  * **Owners** (Full control)
  * **Members** (Can edit)
  * **Visitors** (Read only)
* The site immediately appears in the Sites list.


---

## ?? What Users Can Do

After creation, users can:
? Upload and share documents
? Create Lists and Pages
? Add web parts (News, Quick Links, Documents)
? Manage permissions for teams and visitors

---

## ?? Modern vs Classic

Modern Team Sites:

* Well-optimized for end users
* Faster and mobile friendly
* Uses modern pages and web parts

Classic sites (old design):

* Not recommended for new collaboration scenarios
* Limited modern features



---

## Difference Between Modern Team Site and Communication Site (Summary)

### Modern Team Site

* Used for **team collaboration**
* Multiple users work together
* Members can **edit and upload content**
* Has Owners, Members, Visitors
* Best for **projects, departments, daily work**

### Communication Site

* Used for **broadcasting information**
* Few editors, many readers
* Content is mostly **read-only for users**
* Best for **announcements, news, portals**

**Short difference:**
Modern Team Site = collaborate
Communication Site = communicate

---

## How to Create a Modern Team Site (Step by Step)

### Using SharePoint Central Administration

1. Open **SharePoint Central Administration**
2. Go to **Application Management**
3. Click **Create site collections**
4. Select the required **Web Application**
5. Under **Template Selection**, choose
   **Team Site (Modern Experience)**
6. Enter:

   * Title
   * URL
   * Primary Site Collection Administrator
7. Click **OK**

The Modern Team Site is now created.

---

## How to Upload Content to a Modern Team Site

### Uploading Documents

1. Open the Modern Team Site in the browser
2. Click **Documents**
3. Click **Upload**
4. Choose **Files** or **Folder**
5. Select the file from your computer
6. Click **Open**

The file is uploaded and stored in SQL Server.

---

### Creating a Page

1. Open the Modern Team Site
2. Click **New**
3. Select **Page**
4. Choose a page layout
5. Enter page title
6. Add content using web parts
7. Click **Publish**

---

### Creating a List

1. Open the Modern Team Site
2. Click **New**
3. Select **List**
4. Choose:

   * Blank list
   * From Excel
   * From existing list
5. Enter list name
6. Click **Create**

---

## How It Works (Simple Flow)

1. User uploads or views content
2. Request goes to Front End Server
3. Front End Server communicates with Backend (SQL)
4. Data is saved or retrieved
5. Content is shown to the user

---

## Final One-Line Summary

Modern Team Site is created for collaboration, users upload and manage content, Front End Servers display it, and SQL Server stores it.
